# 시작하기

## Step 1: 메트로 서버 시작

```bash
# using npm
npm start

# OR using Yarn
yarn start
```

## Step 2: Start your Application

choose android or ios

### Android

```bash
# using npm
npm run android

# OR using Yarn
yarn android
```

### iOS

```bash
# using npm
npm run ios

# OR using Yarn
yarn ios
```
